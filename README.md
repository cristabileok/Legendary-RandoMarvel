# Legendary-RandoMarvel
Marvel Legendary Randomizer for Android

Simple and easy-to-use Marvel Legendary Randomizer.

Doesn't require an Internet connection, so you can set up your game anywhere!

Select the Number of Players to automatically create a Setup.

<img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184555967-c52ee65d-24b6-4043-baad-0b3734cee218.jpeg" width=300 height=511 />  <img alt="Legendary RandoMarvel cap2" src="https://user-images.githubusercontent.com/98287876/184555968-dbcea585-134e-4c00-b2fa-3cf12cd185d6.jpeg" width=300 height=511 />

Select a particular category to randomize it on its own, or to randomize again a result you didn't like.

<img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184556028-76829e1c-8463-4d6e-af10-7f5f377aae42.jpeg" width=300 height=511 />

Press on the Results to get more info:

- If you press on a Scheme, you can see its description and keywords. If the Scheme has a Transform or Unveil ability, you can use it from here!

<img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184556034-d6fa6cee-b8f2-4c18-aadd-60d027248611.jpeg" width=300 height=511 /> <img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184556048-975a18c8-02a9-487a-9a1e-35078601e3e9.jpeg" width=300 height=511 />

- If you press on a Mastermind, Villains, Hechmen o Heroes category, you can see their keywords.

<img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184556052-62cac1a9-1b04-4ec4-9c30-f7bba115cf30.jpeg" width=300 height=511 />

Use the Search Function to search for Schemes, Mastermind, Villains, Henchmen, Heroes or Keywords!

<img alt="Legendary RandoMarvel cap1" src="https://user-images.githubusercontent.com/98287876/184556057-837d9462-3e96-4010-8171-6126307e0b58.jpeg" width=300 height=511 />

You can save a Game Setup for later, even if you close the app or turn down your phone!
