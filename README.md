# Legendary-RandoMarvelizer
Marvel Legendary Randomizer for Android

Simple and easy-to-use Marvel Legendary Randomizer.

Doesn't require an Internet connection, so you can set up your game anywhere!

<img alt="RandoMarvelizer cap1" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%201.jpeg" width=300 height=511 />

Select the Number of Players to automatically create a Setup.

<img alt="RandoMarvelizer cap3" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%203.jpeg" width=300 height=511 />  <img alt="RandoMarvelizer cap4" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%204.jpeg" width=300 height=511 />

Select a particular category to randomize it on its own, or to randomize again a result you didn't like.

<img alt="RandoMarvelizer cap2" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%202.jpeg" width=300 height=511 /> <img alt="RandoMarvelizer cap7" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%207.jpeg" width=300 height=511 />

Add more Masterminds, Villains, Henchmen or Heroes if needed.
Remove those you don't like, or randomize an entire category altogether.

<img alt="RandoMarvelizer cap5" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%205.jpeg" width=300 height=511 /> <img alt="RandoMarvelizer cap6" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%206.jpeg" width=300 height=511 />

<b>Search Function</b>

Use the Search Function to search for Schemes, Mastermind, Villains, Henchmen, Heroes or Keywords.
You can see their descriptions or keywords, and add them to the current game setup.

<img alt="RandoMarvelizer cap12" src="https://raw.githubusercontent.com/cristabileok/Legendary-RandoMarvel/main/screen%20captures/RandoMarvelizer%20Cap%2012.jpeg" width=300 height=511 />

<b>Save and Load Games</b>

You can save a Game Setup for later, even if you close the app or turn off your phone!
